# SatyaJiwa

Platform kesehatan mental untuk konsultasi online, latihan mandiri, dan edukasi. Kelola stres, kecemasan, dan emosi dengan privasi terjamin.

## Fitur Utama
- **Konsultasi Online**: Video call/chat gratis pertama dengan psikolog/psikiater, 24/7.
- **Latihan Mandiri**: Meditasi, mindfulness, breathing exercise dengan tutorial.
- **Mood Journal**: Tulis jurnal harian, analisis otomatis kondisi mental.
- **Gamifikasi**: Poin dan badges untuk motivasi latihan.
- **Edukasi**: Artikel/video seperti "5 Cara Mengurangi Stres".
- **Penilaian**: Testimoni pengguna.
- **Penawaran**: Konsultasi gratis, akses premium, enkripsi end-to-end.
